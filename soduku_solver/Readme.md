Input file is the puzzle in CSV format. The zeros can be replaced with spaces for blank cells in the puzzle.

Run:
python solve_soduku.py puzzle.txt

for eg:

,,,,,,,0,0
0,0,3,6,0,0,0,0,0
0,7,0,0,9,0,2,0,0
0,5,0,0,0,7,0,0,0
0,0,0,0,4,5,7,0,0
0,0,0,1,0,0,0,3,0
0,0,1,0,0,0,0,6,8
0,0,8,5,0,0,0,1,0
0,9,0,0,0,0,4,0,0
